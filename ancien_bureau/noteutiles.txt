nslookup > set type=mx est utilisée pour effectuer une recherche DNS spécifique sur les enregistrements MX (Mail Exchanger) d'un domaine. Lorsque vous exécutez cette commande, vous ouvrez l'outil de ligne de commande "nslookup" et vous définissez le type de recherche sur les enregistrements MX en utilisant l'option "set type=mx". Cela permet d'obtenir les informations sur les serveurs de messagerie associés au domaine spécifié.

nslookup > set type=soa est utilisée pour effectuer une recherche DNS spécifique sur l'enregistrement SOA (Start of Authority) d'un domaine. Lorsque vous exécutez cette commande, vous ouvrez l'outil de ligne de commande "nslookup" et vous définissez le type de recherche sur l'enregistrement SOA en utilisant l'option "set type=soa". Cela permet d'obtenir des informations sur l'autorité du domaine, y compris le nom du serveur de noms primaire et l'adresse e-mail de l'administrateur du domaine.

nslookup > set type=ns est utilisée pour effectuer une recherche DNS spécifique sur les enregistrements de serveurs de noms (NS) d'un domaine. Lorsque vous exécutez cette commande, vous ouvrez l'outil de ligne de commande "nslookup" et vous définissez le type de recherche sur les enregistrements NS en utilisant l'option "set type=ns". Cela permet d'obtenir la liste des serveurs de noms autoritaires associés au domaine, qui sont responsables de la résolution des noms de domaine pour ce domaine spécifique.
